<h1>Only in Microsoft Windows</h1>

<h3>Requirements:</h3>
- Python 3.8+
- Pillow (pip install pillow)

<h3>PYTHON</h3>
Download & Install python form python.org

<h3>Library Pillow</h3>
pip install pillow

<h3>Run PY</h3>
python ImageAboveAll.py
